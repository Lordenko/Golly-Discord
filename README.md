# Install

## Stage 1: Install python3.13

## Stage 2: Create and activate venv
```powershell
python3.13 -m venv myenv
source myenv/bin/activate
```

## Stage 3: Install packages
```powershell
pip install -r requirements.txt
```

## Stage 4: Create .env file
```txt
TOKEN=TOKEN
```

## Stage 5: Start bot
```powershell
python3.13 bot.py
```
